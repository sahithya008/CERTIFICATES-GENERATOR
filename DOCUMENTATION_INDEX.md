## üí° Key Concepts

### Analytics Service
The `AnalyticsService` class handles all data aggregation and provides 12+ methods for different analytics queries.

### API Endpoints
Four Flask routes that expose analytics data in JSON format, with support for date range filtering.

### Dashboard UI
Enhanced admin dashboard with 5 interactive charts, metric cards, and filtering capabilities.

### Data Export
CSV export functionality for reporting and external analysis.

### Security
Admin authentication required on all endpoints with proper session management.

**Q: How do I use the feature?**
A: Login to admin panel, click Analytics tab, explore charts and use filters.

**Q: How do I export data?**
A: Click the "Export CSV" button on the Analytics tab.

**Q: How do I filter by date?**
A: Select start and end dates, then click the "Filter" button.

**Q: Is it secure?**
A: Yes, all endpoints require admin authentication with proper session validation.
